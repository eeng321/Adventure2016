Installation Instructions
	1. Create new build directory. ie. mkdir build
	2. cd build
	3. cmake -G "Unix Makefiles" -DCMAKE_BUILD_TYPE=Release <path to adventure2016> 
	4. make
	5. binaries located in bin/
